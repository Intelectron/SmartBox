# SmartBox
 This is a pre-alpha prototype made on prefboard so that you can make it easily at home with few components.
## List Of Components
  * Arduino Uno x 1
  * HC-05 Bluetooth Module x 1
  * 12V SPST Relay x 2
  * 2N3904(BJT) x 2
  * Leds x 2
  * 1kohm resistor  x 1
  * 560ohm resistor x 2
  * 1N4007 diode  x 2
  * Bar strips for making the shield
  * Prefboard
